# MySQL Data Manipulation Language (DML)

Using the `classicmodels` database schema.

---

# + Insert Data

## 1. INSERT INTO

Used to insert a single row into a table.

### Syntax
```sql
INSERT INTO table_name(column1, column2)
VALUES(value1, value2);
```

### Example
```sql
INSERT INTO offices
(
    officeCode,
    city,
    phone,
    addressLine1,
    country,
    postalCode,
    territory
)
VALUES
(
    '8',
    'Phnom Penh',
    '+855 12 345 678',
    'Street 2004',
    'Cambodia',
    '12000',
    'Asia'
);
```

## 3. INSERT INTO SELECT

Copy data from one table into another table.

### Syntax
```sql
INSERT INTO new_table(column1, column2)
SELECT column1, column2
FROM old_table;
```

### Example
```sql
CREATE TABLE customer_backup
AS
SELECT *
FROM customers
WHERE 1 = 0;
```

```sql
INSERT INTO customer_backup
SELECT *
FROM customers
WHERE country = 'USA';
```

---

## 4. INSERT ON DUPLICATE KEY UPDATE

If duplicate key exists, update the existing row.

### Syntax
```sql
INSERT INTO table_name(...)
VALUES(...)
ON DUPLICATE KEY UPDATE
column_name = value;
```

### Example
```sql
INSERT INTO productlines
(
    productLine,
    textDescription
)
VALUES
(
    'Motorcycles',
    'Updated motorcycle description'
)
ON DUPLICATE KEY UPDATE
textDescription = 'Updated motorcycle description';
```

---

## 5. INSERT IGNORE

Ignore duplicate key errors.

### Example
```sql
INSERT IGNORE INTO productlines
(
    productLine,
    textDescription
)
VALUES
(
    'Classic Cars',
    'Duplicate row test'
);
```

# + Update Data

## 1. UPDATE

Used to modify existing data.

### Syntax
```sql
UPDATE table_name
SET column_name = value
WHERE condition;
```

### Example
```sql
UPDATE customers
SET city = 'Phnom Penh'
WHERE customerNumber = 103;
```

---

## 2. UPDATE JOIN

Update a table using another table.

### Syntax
```sql
UPDATE table1
JOIN table2
ON table1.id = table2.id
SET table1.column_name = value;
```

### Example
```sql
UPDATE customers c
JOIN employees e
ON c.salesRepEmployeeNumber = e.employeeNumber
SET c.creditLimit = 150000
WHERE e.jobTitle = 'Sales Rep';
```

---

# + Delete Data

## 1. DELETE JOIN

Delete rows using JOIN.

### Syntax
```sql
DELETE table1
FROM table1
JOIN table2
ON table1.id = table2.id
WHERE condition;
```

### Example
```sql
DELETE p
FROM payments p
JOIN customers c
ON p.customerNumber = c.customerNumber
WHERE c.country = 'France';
```

---

## 2. ON DELETE CASCADE

Automatically delete child rows when parent row is deleted.

### Example

### Parent Table
```sql
CREATE TABLE departments
(
    department_id INT PRIMARY KEY,
    department_name VARCHAR(100)
);
```

### Child Table
```sql
CREATE TABLE employees_test
(
    employee_id INT PRIMARY KEY,
    employee_name VARCHAR(100),
    department_id INT,
    FOREIGN KEY(department_id)
    REFERENCES departments(department_id)
    ON DELETE CASCADE
);
```

### Insert Data
```sql
INSERT INTO departments
VALUES
(1, 'IT');
```

```sql
INSERT INTO employees_test
VALUES
(101, 'John', 1);
```

### Delete Parent Row
```sql
DELETE FROM departments
WHERE department_id = 1;
```

The related employee row will also be deleted automatically.

---

## 3. TRUNCATE TABLE

Remove all rows from a table quickly.

### Syntax
```sql
TRUNCATE TABLE table_name;
```

### Example
```sql
TRUNCATE TABLE customer_backup;
```
