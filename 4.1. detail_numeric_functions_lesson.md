# Numeric Functions (ClassicModels)

## Lesson Overview

This lesson introduces MySQL numeric functions with practical examples
using the **classicmodels** database.

------------------------------------------------------------------------

## 1. Basic Arithmetic Functions

### ABS()

Returns absolute value.

``` sql
SELECT ABS(-500) AS result;
```

### SIGN()

Returns -1, 0, or 1.

``` sql
SELECT SIGN(-10), SIGN(0), SIGN(25);
```

------------------------------------------------------------------------

## 2. Rounding Functions

### ROUND()

``` sql
SELECT ROUND(123.456, 2);
```

``` sql
SELECT customerName, ROUND(creditLimit, 0) AS rounded_credit
FROM customers;
```

### CEIL() / CEILING()

``` sql
SELECT CEIL(123.1);
```

``` sql
SELECT productName, CEIL(buyPrice) AS price_up
FROM products;
```

### FLOOR()

``` sql
SELECT FLOOR(123.9);
```

### TRUNCATE()

``` sql
SELECT TRUNCATE(123.4567, 2);
```

------------------------------------------------------------------------

## 3. Power & Roots

### POWER()

``` sql
SELECT POWER(2, 3);
```

### SQRT()

``` sql
SELECT SQRT(64);
```

------------------------------------------------------------------------

## 4. MOD()

``` sql
SELECT MOD(10, 3);
```

``` sql
SELECT orderNumber, MOD(orderNumber, 2) AS odd_even
FROM orders;
```

------------------------------------------------------------------------

## 5. RAND()

``` sql
SELECT RAND();
```

``` sql
SELECT productName
FROM products
ORDER BY RAND()
LIMIT 5;
```

------------------------------------------------------------------------

## 6. FORMAT()

``` sql
SELECT FORMAT(1234567.89, 2);
```

``` sql
SELECT customerName, FORMAT(creditLimit, 2)
FROM customers;
```

------------------------------------------------------------------------

## 7. GREATEST() / LEAST()

``` sql
SELECT GREATEST(10, 20, 5);
SELECT LEAST(10, 20, 5);
```

------------------------------------------------------------------------

## 8. Aggregate Functions

### SUM()

``` sql
SELECT SUM(amount) FROM payments;
```

### AVG()

``` sql
SELECT AVG(amount) FROM payments;
```

### MAX() / MIN()

``` sql
SELECT MAX(amount), MIN(amount) FROM payments;
```

------------------------------------------------------------------------

## 9. Combined Example

``` sql
SELECT 
    customerNumber,
    ROUND(AVG(amount), 2) AS avg_payment,
    MAX(amount) AS max_payment,
    MIN(amount) AS min_payment
FROM payments
GROUP BY customerNumber;
```
