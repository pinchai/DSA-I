# MySQL Grouping Functions

Grouping functions are aggregate functions used to calculate values from multiple rows and return a single result.

Common grouping functions in MySQL:

| Function | Description |
|---|---|
| `COUNT()` | Counts rows |
| `SUM()` | Adds values |
| `AVG()` | Calculates average |
| `MAX()` | Finds highest value |
| `MIN()` | Finds lowest value |

---

# 1. COUNT()

Counts the number of rows.

```sql
SELECT COUNT(*) AS total_customers
FROM customers;
```

Count customers from USA:

```sql
SELECT COUNT(*) AS usa_customers
FROM customers
WHERE country = 'USA';
```

---

# 2. SUM()

Adds numeric values together.

Calculate total payments:

```sql
SELECT SUM(amount) AS total_payment
FROM payments;
```

Total quantity ordered:

```sql
SELECT SUM(quantityOrdered) AS total_quantity
FROM orderdetails;
```

---

# 3. AVG()

Finds the average value.

Average payment amount:

```sql
SELECT AVG(amount) AS average_payment
FROM payments;
```

Average product price:

```sql
SELECT AVG(buyPrice) AS average_price
FROM products;
```

---

# 4. MAX()

Returns the highest value.

Highest payment:

```sql
SELECT MAX(amount) AS highest_payment
FROM payments;
```

Most expensive product:

```sql
SELECT MAX(MSRP) AS highest_price
FROM products;
```

---

# 5. MIN()

Returns the lowest value.

Lowest payment:

```sql
SELECT MIN(amount) AS lowest_payment
FROM payments;
```

Cheapest product:

```sql
SELECT MIN(buyPrice) AS cheapest_product
FROM products;
```

---

# GROUP BY with Grouping Functions

`GROUP BY` is used to group rows that have the same values.

Total customers by country:

```sql
SELECT country, COUNT(*) AS total_customers
FROM customers
GROUP BY country;
```

Average payment by customer:

```sql
SELECT customerNumber, AVG(amount) AS average_payment
FROM payments
GROUP BY customerNumber;
```

Total sales per product:

```sql
SELECT productCode, SUM(quantityOrdered) AS total_sold
FROM orderdetails
GROUP BY productCode;
```

---

# HAVING Clause

`HAVING` filters grouped data.

Countries with more than 5 customers:

```sql
SELECT country, COUNT(*) AS total_customers
FROM customers
GROUP BY country
HAVING COUNT(*) > 5;
```

Customers with total payments greater than 100000:

```sql
SELECT customerNumber, SUM(amount) AS total_paid
FROM payments
GROUP BY customerNumber
HAVING SUM(amount) > 100000;
```
