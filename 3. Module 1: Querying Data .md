# Module 1: Querying Data (MySQL)

## Module Objective

By the end of this module, students will be able to retrieve data from
the classicmodels database using SELECT queries and apply filtering,
sorting, and conditions.

## Topics Covered

-   SELECT
-   WHERE
-   ORDER BY
-   DISTINCT
-   AND / OR
-   IN
-   BETWEEN
-   LIKE
-   LIMIT
-   IS NULL
-   Aliases

------------------------------------------------------------------------

## 1. SELECT Statement

The SELECT statement is used to retrieve data from one or more columns
in a table.

``` sql
SELECT customerName, country
FROM customers;
```

------------------------------------------------------------------------

## 2. WHERE Clause

Filters rows based on conditions.

``` sql
SELECT customerName, country
FROM customers
WHERE country = 'USA';
```

------------------------------------------------------------------------

## 3. ORDER BY

Sorts the result set.

``` sql
SELECT productName, buyPrice
FROM products
ORDER BY buyPrice DESC;
```

------------------------------------------------------------------------

## 4. DISTINCT

Removes duplicate values.

``` sql
SELECT DISTINCT country
FROM customers;
```

------------------------------------------------------------------------

## 5. AND / OR

Combines multiple conditions.

``` sql
SELECT customerName, creditLimit
FROM customers
WHERE country = 'USA' AND creditLimit > 50000;
```

------------------------------------------------------------------------

## 6. IN

Matches values from a list.

``` sql
SELECT customerName
FROM customers
WHERE country IN ('USA', 'France', 'UK');
```

------------------------------------------------------------------------

## 7. BETWEEN

Filters values in a range.

``` sql
SELECT customerName, creditLimit
FROM customers
WHERE creditLimit BETWEEN 30000 AND 90000;
```

------------------------------------------------------------------------

## 8. LIKE

Searches patterns.

``` sql
SELECT customerName
FROM customers
WHERE customerName LIKE '%Ltd';
```

------------------------------------------------------------------------

## 9. LIMIT

Limits returned rows.

``` sql
SELECT productName, MSRP
FROM products
ORDER BY MSRP DESC
LIMIT 5;
```

------------------------------------------------------------------------

## 10. IS NULL

Checks missing values.

``` sql
SELECT orderNumber, shippedDate
FROM orders
WHERE shippedDate IS NULL;
```

------------------------------------------------------------------------

## 11. Aliases

Renames columns.

``` sql
SELECT customerName AS Customer, creditLimit AS Credit
FROM customers;
```

------------------------------------------------------------------------
