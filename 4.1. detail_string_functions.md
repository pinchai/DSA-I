# MySQL String Functions

## 1. CONCAT() – Combine Strings
```sql
SELECT CONCAT('Hello', ' ', 'World') AS result;
```

```sql
SELECT CONCAT(firstName, ' ', lastName) AS full_name
FROM customers;
```

---

## 2. UPPER() / LOWER() – Change Case
```sql
SELECT UPPER('chai');   -- CHAI
SELECT LOWER('MYSQL');  -- mysql
```

---

## 3. LENGTH() / CHAR_LENGTH() – String Length
```sql
SELECT LENGTH('Hello');        -- bytes
SELECT CHAR_LENGTH('Hello');   -- characters
```

---

## 4. SUBSTRING()
```sql
SELECT SUBSTRING('Hello World', 1, 5); -- Hello
```

---

## 5. REPLACE()
```sql
SELECT REPLACE('Hello World', 'World', 'MySQL');
```

---

## 6. TRIM()
```sql
SELECT TRIM('   Hello   ');
SELECT LTRIM('   Hello');
SELECT RTRIM('Hello   ');
```

---

## 7. LEFT() / RIGHT()
```sql
SELECT LEFT('Hello World', 5);
SELECT RIGHT('Hello World', 5);
```

---

## 8. LOCATE()
```sql
SELECT LOCATE('World', 'Hello World');
```

---

## 9. INSTR()
```sql
SELECT INSTR('Hello World', 'World');
```

---

## 10. LPAD() / RPAD()
```sql
SELECT LPAD('5', 3, '0');
SELECT RPAD('5', 3, '0');
```

---

## 11. REVERSE()
```sql
SELECT REVERSE('Hello');
```

---

## 12. FORMAT()
```sql
SELECT FORMAT(1234567.89, 2);
```

---

## 13. ASCII() / CHAR()
```sql
SELECT ASCII('A');
SELECT CHAR(65);
```

---

## Practical Example
```sql
SELECT 
    customerName,
    UPPER(country) AS country_upper,
    CONCAT(city, ', ', country) AS full_address,
    LENGTH(customerName) AS name_length
FROM customers;
```
