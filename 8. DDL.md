# MySQL Managing Tables

## Create Tables

```sql
CREATE TABLE students (
    id INT PRIMARY KEY,
    name VARCHAR(100),
    age INT
);
```

## AUTO_INCREMENT

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50)
);
```

## Generated Columns

```sql
CREATE TABLE products (
    price DECIMAL(10,2),
    quantity INT,
    total DECIMAL(10,2) GENERATED ALWAYS AS (price * quantity) STORED
);
```
